# qt-file-encryptor

一个简单的加密文件的工具。在key的框里输入一个十进制数，可生成指定位数的随机字节，加密时会将原字节加上key中字节的数值。
 A simple tool for encrypting files

学习了设定可执行文件自身的图标、以及定制化标题栏。
尝试了使用qss制作不同的显示主题。
